# Game data

## `keen1.jsdos` — included (Keen 1 shareware v1.31)

This bundle contains the **Commander Keen 1 shareware** episode ("Marooned on Mars", v1.31),
which Apogee/id Software released as **freely redistributable**. It is packaged here as a
`.jsdos` bundle:

```
.jsdos/dosbox.conf   DOSBox config (autoexec runs KEEN1.EXE)
EGAHEAD.CK1   EGALATCH.CK1   EGASPRIT.CK1   \
LEVEL01.CK1 … LEVEL90.CK1                    > Keen 1 shareware data files
SOUNDS.CK1  FINALE.CK1  CTLPANEL.CK1  …      /
KEEN1.EXE            Keen 1 shareware executable
```

Keen 1 ("Invasion of the Vorticons", episode one) is the shareware chapter of the trilogy and may
be freely distributed.

## Keen 2 & 3 — NOT included (commercial)

Episodes 2 ("The Earth Explodes") and 3 ("Keen Must Die!") are commercial games.
**Their data files must never be committed to this repository.** Players supply their own copies
at runtime via the file picker on the site; those files are assembled into a `.jsdos` bundle in
the browser and never uploaded anywhere.

Buy them: [GOG](https://www.gog.com/game/commander_keen_complete_pack) ·
[Steam](https://store.steampowered.com/app/9180/Commander_Keen_Pack/).
